### âœ… **4. Debugging in VS Code**

To enable debugging in **VS Code**, add the following configuration in `.vscode/launch.json`:

```json
{
  "configurations": [
    {
      "name": "FastAPI Debug",
      "type": "python",
      "request": "launch",
      "module": "uvicorn",
      "args": ["main:app", "--host", "127.0.0.1", "--port", "8000", "--reload"]
    }
  ]
}
```

Now, press **F5** to start debugging.
