# Mini Project:

## **1. Simple To-Do API (Basic CRUD Operations)**

This To-Do API allows users to:

- **Create a new task** (`POST /tasks/`)
- **Retrieve all tasks** (`GET /tasks/`)
- **Update a task** (`PUT /tasks/{task_id}`)
- **Delete a task** (`DELETE /tasks/{task_id}`)
